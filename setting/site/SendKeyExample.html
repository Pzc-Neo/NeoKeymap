简介

输入文本或按键的功能有啥用呢 ?

  1. 输入成对符号并移动光标, 比如「 ... 」(...)
  2. 常用文本缩写, 比如可以让 yx 等于 youxiang@mail.com
  3. 做一些简单的 「 按键自动化 」, 比如按一下分号键并输入 ver 就能查看系统版本
  4. 在阅读下面的文档前, 可以先看下分号缩写 gg、sk、ver 是怎么配置的

 

例子

用 <font> 标签包裹选中的文本

​x
^x           ; 发送 Ctrl+X
sleep 100    ; 等 100 ms, 等剪切板拿到复制的文本


; {text} 前缀表示以文本模式发送,  这不会被输入法影响,  也不会把 {enter} 解析为回车键
{text}<span alt="underline"></span>


{left 7}^v    ; 移动光标、发送 Ctrl+V

 

输入「 abcde🐶 」并复制粘贴两次

xxxxxxxxxx
abcd🐶                  ; 输入 abcd🐶
sleep 100                ; 等待 100 ms
+{home}                  ; 输入 Shift+Home 选中当前行
sleep 100                ; 等待 100 ms
^c{end}{enter}           ; 输入 Ctrl+C ➤ End ➤ Enter
sleep 100                ; 等待 100 ms
^v{space 4}^v            ; 输入 Ctrl+V ➤ 四下空格 ➤ Ctrl+V
